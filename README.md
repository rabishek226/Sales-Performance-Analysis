# Sales-Performance-Analysis

**Project Summary**
I transformed raw sales data into an interactive dashboard to help a coffee shop identify trends and top-performing products. The project covers the full data analysis lifecycle: Data Cleaning, Processing, and Visualization.

**Key Features**
1 . Interactive Date Analysis: Integrated a Date Slicer (Timeline) to dynamically filter and view sales trends across specific months and years.
2 . Dynamic Dashboard: Built with Slicers for filtering by Country and Product Type.
3 . Customer Analysis: Identified the Top 5 Customers per country and product line.
4 . Data Cleaning: Standardized formats and removed duplicates using advanced Excel functions.

**Tech Stack**
1 . Microsoft Excel 
2 . Advanced Formulas: XLOOKUP, SUMIFS, INDEX-MATCH, IF
3 . Pivot Tables & Pivot Charts
